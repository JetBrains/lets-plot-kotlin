# LaTeX Support

Lets-Plot has limited support for the LaTeX formulas. See examples below.

## Superscript and Subscript

``2^5`` → 2<sup>5</sup>

``2^{10}`` → 2<sup>10</sup>

``2^{-5}`` → 2<sup>−5</sup>

``x_i`` → x<sub>i</sub>

``x_{min}`` → x<sub>min</sub>

## Spacing

``A B`` → AB

``A \quad B`` → A B

``A \qquad B`` → A  B

``A \, B`` → A B

``A \: B`` → A B

``A \  B`` → A B
